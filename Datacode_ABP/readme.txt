This folder is the original code for "Non Line of Light Imaging with Absorption Backproject", which includes the BP, ABP, FBP, EBP, and fk algorithms in the paper. Run the following main program to obtain the data in the paper, and run the code in the figure folder to obtain the images in the paper.
reconstruction_ABP.m
reconstruction_BP.m
reconstruction_EBP.m
reconstruction_FBP.m
reconstruction_FK.m
reconstruction_ncBP.m
reconstruction_non_retroreflective.m
The Dragon180min and Statue180min datasets can be downloaded at the following link:
https://drive.google.com/drive/folders/115td2Vr6NBUpr4WWRpuWvWyylpHvzbSr?usp=sharing
